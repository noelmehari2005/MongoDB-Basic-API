# MongoDB-Basic-API

1) the .env file stores app settings and info separate from the code
2) The code checks if the user wants a lowest price or highest price, then only shows items in that price range.
3) seed.js is used to fill the database with starter data.
4) Put changes or updates the whole value, while Patch only changes parts of the value.
